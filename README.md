Weather App   
Functionality:
**The App loads current weather details as per the city entered using OpenWeatherMap API.**     
**Details Shown:**    
-Current Temperature    
-Sunrise, Sunset    
-Pressure    
-Visibility    
-Humidity   
-Variance in Temperature   
-Wind Speed  
-Latitude, Longitude    

**Libraries Used**   

**-Retrofit 2.0**        
**-Airbnb Lottie**
