Weather App
Functionality:
**The App loads current weather details as per the city entered.**     
**Details:**   
- Current Temperature    
-Sunrise, Sunset    
-Pressure    
-Visibility    
-Humidity   
-Variance in Temperature   
-Wind Speed  
-Latitude, Longitude
